 ## StepUp
```
git clone https://github.com/tooggle/License-plate-identification.git
```
create conda env
```
conda create -n yolov7
```
Enter the environment
```
conda activate yolov7
```
Install dependency
```
pip3 install -r requirements.txt
```
Run **Streamlit**
```
streamlit run app.py
```
